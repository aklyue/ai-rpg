# AI-RPG

Мобильная ролевая игра (RPG), использующая технологии искусственного интеллекта для динамической генерации уникальных сюжетов, квестов и игрового контента в реальном времени.

## 🚀 Особенности проекта

* **Динамический сюжет:** Интеграция с моделями ИИ для создания уникальных сценариев и диалогов на основе выборов игрока.
* **Современный стек:** Полная типизация и высокая производительность благодаря TypeScript и React Native.
* **Управление состоянием:** Предсказуемая и чистая архитектура данных с использованием Redux Toolkit.
* **Кроссплатформенность:** Быстрая сборка и запуск на iOS и Android через экосистему Expo.

## 🛠 Технологический стек

* **Frontend:** React Native (Expo), TypeScript, Redux Toolkit
* **Backend API:** ExpressJS, Node.js
* **AI Integration:** OpenAI API / Другие LLM-провайдеры

## 📂 Структура репозитория

```text
├── assets/             # Графические ресурсы, иконки и шрифты
├── src/                # Исходный код приложения
│   ├── components/     # Переиспользуемые UI-компоненты
│   ├── screens/        # Экраны приложения (меню, игра, боевая система)
│   ├── store/          # Логика Redux (slices, селекторы)
│   └── services/       # Интеграция с API и ИИ-сервисами
├── App.tsx             # Главный компонент входа
├── app.json            # Конфигурация Expo-приложения
└── eas.json            # Настройки сборки Expo Application Services
```

## ⚙️ Быстрый старт

### Требования
Убедитесь, что у вас установлены [Node.js](https://nodejs.org) (версии 18+) и менеджер пакетов `npm` или `yarn`.

### Установка зависимостей
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd ai-rpg
   ```

2. Установите необходимые пакеты:
   ```bash
   npm install
   ```

### Запуск в среде разработки

Для запуска проекта локально через Expo CLI выполните команду:
```bash
npm start
```

После этого вы сможете:
* Открыть проект на физическом устройстве через приложение **Expo Go**, отсканировав QR-код.
* Запустить проект в эмуляторе Android (`a`) или симуляторе iOS (`i`).

## 🛠 Сборка приложения

Проект использует инструмент **EAS Build** для компиляции нативных пакетов. Для сборки выполните:

```bash
# Для Android
eas build --platform android

# Для iOS
eas build --platform ios
```
